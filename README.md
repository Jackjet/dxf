# dvf v1.0.0

> 数据可视化工厂（Data Visualization Factory）

# 项目结构
  ├─ 主界面
  ├─ 主题管理／
  ├   ├─ 主题上传
  ├   ├─ 主题信息
  ├   └─ 主题视图
  ├─ 性能监测
  ├─ 系统日志
  └─ 控制台

#### Build Setup

``` bash

# install dependencies
> yarn 
#（or npm install)

# serve with hot reload at localhost:9080
> yarn run dev 
#(or npm run dev)

# build electron application for production
> yarn run build 
#(or npm run build)

# run unit tests
> yarn test 
#(or npm test)

# lint all JS/Vue component files in `src/`
> yarn run lint 
#(or npm run lint)

```
