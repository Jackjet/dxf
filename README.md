# dxf v1.0.0

> 数据可视化工厂管理后台（Data Visualization Factory）

# 项目结构
  ├─ 设备面板
  ├─ 账号管理
  ├─ 数据管理
  ├─ 视频管理
  ├─ 日志信息
  └─ 系统设置

#### Build Setup

``` bash

# install dependencies
> yarn 
#（or npm install)

# serve with hot reload at localhost:9080
> yarn run dev 
#(or npm run dev)

# build electron application for production
> yarn run build 
#(or npm run build)

# run unit tests
> yarn test 
#(or npm test)

# lint all JS/Vue component files in `src/`
> yarn run lint 
#(or npm run lint)

```
